Build on Tester.io.

A clean REST + WebSocket API for market data, accounts, and order execution. Get a key, make your first request in minutes.

Quickstart

Authenticate with a bearer token, then call the REST API. Every response is JSON.

# Fetch the current TST market
curl https://api.tester.io/v1/markets/TST-USD \
  -H "Authorization: Bearer $TESTER_API_KEY"

Explore the API

Authentication

Create and rotate API keys, scope permissions, and sign requests.

Market data

REST snapshots and WebSocket streams for prices, depth, and trades.

Orders

Place, modify, and cancel leveraged orders with idempotency keys.

Conventions

  • Base URL: https://api.tester.io/v1
  • All timestamps are ISO-8601 UTC.
  • Rate limits are returned in the X-RateLimit-* headers.
  • Errors use standard HTTP status codes with a JSON error body.
