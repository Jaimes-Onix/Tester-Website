Abstract

Tester.io is a platform for leveraged exposure to any token, built on self-custody rails. This paper describes the execution model, the risk framework that protects users, and the TST token that aligns the network's participants.

The problem

Leverage has historically meant trusting an opaque intermediary with your assets. Tester.io removes that trade-off: users keep custody while still accessing fast, predictable execution and transparent risk views.

Protocol design

  • Deterministic order pipeline tuned for low, steady latency.
  • On-chain settlement with off-chain matching for speed.
  • Continuous liquidation engine with conservative, transparent margins.
  • Open APIs so developers can build their own tools on top.

Token economy

The TST token coordinates fees, governance, and incentives across the network. Full supply and distribution figures live on the token metrics page.

Roadmap

  • 2014–2017 — Core definitions and automated execution tools shipped.
  • 2022 — Developer APIs (in progress).
  • 2025+ — Community governance and discussion layer (planned).
